## 获取当前时间<div id='parse_nano'></div>

`context.parse_nano(nano, format)`

**参数**

| 参数       | 类型  | 说明       |
| ---------- | ----- | ---------- |
| nano  | long  | 纳秒时间戳 |
| format | str   | 时间格式(例: %Y%m%d %H:%M:%S) |

**返回**

| 返回 | 类型 | 说明               |
| ---- | ---- | ------------------ |
| timeStr   | str     | 时间字符串 |
