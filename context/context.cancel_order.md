

## `撤单`<div id='cancel_order'></div>

`context.cancel_order(order_id)`

**参数**

| 参数     | 类型 | 说明   |
| -------- | ---- | ------ |
| order_id | long | 订单ID |

**返回**

| 返回      | 类型 | 说明       |
| --------- | ---- | ---------- |
| action_id | long | 订单操作ID |

**范例**

撤销订单 1000111112

`context.cancel_order(1000111112)`
