## 获取当前时间<div id='register_nanotime_callback_with_context'></div>

`context.register_nanotime_callback_with_context(nano, callback)`

**参数**

| 参数       | 类型  | 说明       |
| ---------- | ----- | ---------- |
| nano  | long  | 触发回调的纳秒时间戳 |
| callback | object   | 回调函数(最后一个参数为context) |

**返回**

无
