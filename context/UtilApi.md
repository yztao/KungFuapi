**设置 Log 级别**

`context.set_log_level(level)`

**参数**

| 参数  | 类型 | 说明     |
| ----- | ---- | -------- |
| level | int  | log 级别 |

**返回**

无

**输出 INFO 级别 Log 信息**

`context.log_info(msg)`

**参数**

| 参数 | 类型 | 说明     |
| ---- | ---- | -------- |
| msg  | str  | Log 信息 |

**返回**

无
