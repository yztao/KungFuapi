## 获取当前时间<div id='parse_time'></div>

`context.parse_time(timeStr, format)`

**参数**

| 参数       | 类型  | 说明       |
| ---------- | ----- | ---------- |
| timeStr  | str  | 时间字符串(例: 20190501 09:00:00 |
| format | str   | 时间格式(例: %Y%m%d %H:%M:%S), 需和时间字符串对应 |

**返回**

| 返回 | 类型 | 说明               |
| ---- | ---- | ------------------ |
| nano | long | 纳秒时间戳 |
