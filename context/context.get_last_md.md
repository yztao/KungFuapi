## 获取最后一次行情<div id='get_last_md'></div>

`context.get_last_md(instrument_id, exchange_id)`

**参数**

| 参数          | 类型  | 说明     |
| ------------- | ----- | -------- |
| instrument_id | str   | 合约ID   |
| exchange_id   | str   | 交易所ID |

**返回**

| 返回 | 类型 | 说明               |
| ---- | ---- | ------------------ |
| quote    | Quote     | 行情 |
