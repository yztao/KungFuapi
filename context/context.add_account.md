## 添加交易账号<div id='add_account'></div>

`context.add_account(source_id, account_id, cash_limit)`

**参数**

| 参数       | 类型  | 说明       |
| ---------- | ----- | ---------- |
| source_id  | str   | 行情柜台ID |
| account_id | str   | 账户ID     |
| cash_limit | float | 资金限制   |

**返回**

| 返回   | 类型 | 说明 |
| ------ | ---- | ---- |
| result | bool |      |
