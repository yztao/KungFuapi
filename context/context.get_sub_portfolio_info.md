## 获取策略中账号资产信息<div id='get_sub_portfolio_info'></div>

`context.get_sub_portfolio_info(account_id)`

**参数**

| 参数          | 类型  | 说明     |
| ------------- | :---: | -------- |
| account_id    |  str  | 交易账号 |

**返回**

| 返回           | 类型  | 说明     |
| -------------- | ----- | -------- |
| sub_pnl | SubPortfolioInfo | 账号资产信息 |
