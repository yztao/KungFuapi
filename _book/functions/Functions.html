基本方法

策略可以选择实现以下方法。

init(context)

初始化方法，在启动时候会触发一次。策略使用这个方法来做各种初始化设置。

参数

参数 类型 说明
quote python 对象 策略的全局变量，通过点标记（"."）来获取其属性。

返回

无

范例

on_quote(context,quote)

行情数据的推送会自动触发该方法的调用。

参数

参数 类型 说明
quote Quote 已订阅的合约行情数据

返回

无

范例

on_order(context, order)

订单信息的更新会自动触发该方法的调用。

参数 类型 说明
order Order 策略订单信息

返回

无

范例

on_trade(context, trade)

成交信息的更新会自动触发该方法的调用。

参数 类型 说明
trade Trade 策略成交信息

返回

无
