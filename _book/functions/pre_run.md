## `pre_run(context)`<div id='pre_run'></div>

> 运行前方法。

**参数**

| 参数    | 类型        | 说明                                            |
| ------- | ----------- | ----------------------------------------------- |
| context | python 对象 | 策略的全局变量，通过点标记（"."）来获取其属性。 |

**返回**

无

**范例**

```python
def pre_run(context):
    pass
```
